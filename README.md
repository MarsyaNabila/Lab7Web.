## Portal Berita Marsya - Web Management System (CI4)
Portal Berita Marsya adalah platform manajemen konten (CMS) sederhana yang dirancang untuk mengelola artikel berita seputar dunia teknologi, pemrograman, dan aktivitas akademik. Proyek ini dibangun menggunakan Framework CodeIgniter 4 sebagai bagian dari tugas praktikum pengembangan web.

## Fitur Utama
Sistem ini mencakup dashboard admin fungsional dengan fitur sebagai berikut:

- Manajemen Artikel (CRUD): Menambah, melihat, mengubah, dan menghapus data artikel berita.

- Fitur Pencarian: Memudahkan admin mencari judul artikel tertentu secara cepat melalui search bar di dashboard.

- Pagination: Pembatasan jumlah tampilan data per halaman agar manajemen data lebih rapi dan ringan.

- Upload Gambar: Mendukung pengunggahan gambar unggulan untuk setiap artikel berita.

- UI Responsif: Tampilan antarmuka yang bersih dengan sentuhan warna identitas (Pink/Magenta).

## Teknologi yang Digunakan
- Framework: CodeIgniter 4.x

- Bahasa Pemrograman: PHP

- Database: MySQL

- Frontend: HTML5, CSS3 (Custom Styling), Bootstrap (Optional/Base)

- Tools: Visual Studio Code, XAMPP

## Cara Instalasi

- Clone repositori: ```git clone https://github.com/username-kamu/nama-repo.git```

- Pastikan XAMPP (Apache & MySQL) aktif.

- Import database (file .sql) ke phpMyAdmin.

- Sesuaikan konfigurasi database di file .env atau app/Config/Database.php.

- Jalankan server: ```php spark serve```

- Buka ```localhost:8080``` di browser Anda.
